# meeting-bot

## Installation

```
git clone git@github.com:100mslive/meeting-bot.git
cd meeting-bot
npm i
npm i -g .
```

## Command to run

From any directory, run
```
meeting-bot --url "<meeting_url>"
```

## Help for CLI
```
meeting-bot --help
```

## Examples

### LiveKit

node examples/livekit.js --url "https://meet.livekit.io/rooms/mxqh-7skt" -o output
