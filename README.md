# meeting-bot

## Installation

```
npm i
```

## Command to run

```
node index.js --url "<meeting_url>" -o "<output_dir>" -d "<downloads_dir>"
```

## Help for CLI
```
node index.js --help
```

## Examples

### LiveKit

node examples/livekit.js --url "https://meet.livekit.io/rooms/mxqh-7skt" -o output
